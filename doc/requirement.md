# 大作业要求

## 基本要求

1. 必须基于此开发板。如果使用其他算法，也请在开发板上部署运行；所有算法均可使用。
2. 必须使用两自由度云台，但云台上可以不连接相机（螺丝拆下来保存好，后面要还原）。
3. 如果做物体跟踪，那么云台的跟踪精度、速度都是影响分数的项。

## 加分项

1. 把一个小的、比较浪费时间的功能块用 C 或 C++ 实现，然后编译成 whl 调用。如果实在找不到合适功能块，可以使用 5x5 图像卷积。同时用对应的 Python 实现该功能，用于对比性能。
2. 可以结合网络编程，和云端服务相关联。
3. 可以考虑任务的加速。提醒：Python 的 thread 是一坨翔，不能真正提速。建议使用 pymp。
4. 使用 DNN 时，最后用 RKNN 进行部署。
